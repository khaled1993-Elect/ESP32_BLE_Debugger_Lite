# Beginner Examples

These five examples are designed for the ESP32-S3 and require no external
sensors.

They use:

- GPIO0: onboard BOOT button
- GPIO2: real digital output
- Virtual probes: internal program variables

## Examples

1. Button Controls a Lamp
2. Timed Stair Light
3. Door Open Alarm
4. Temperature Alarm
5. Machine Cycle Monitor

Do not hold BOOT while resetting the board because that enters download mode.
